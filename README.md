## Featured Projects

### [Prime Tracker — Android](https://github.com/FelipiMatheuz/prime-tracker-android)
Inventory and item-tracking app for Warframe, actively maintained since 2023 and reaching version 3.0.12 (build 29).

- **Architecture:** MVVM with explicit layering — `business`, `model`, `service`, `ui`, `viewmodel`
- **Stack:** Kotlin · Jetpack Compose · Navigation · Retrofit · Firebase Auth + Firestore · Credential Manager · Material3
- **Highlights:** Offline-first local state with cloud sync via Firestore; production release builds with R8 shrinking and ProGuard; in-app update flow via Google Play Core; multi-language UI and dynamic theming (Light/Dark)
- **Target SDK:** 36 (latest) · Min SDK: 27

---

### [Palace — Android Card Game](https://github.com/FelipiMatheuz/palace-android)
Open-source Android implementation of the Palace card game, published to Google Play (palace-v2). A portfolio-grade project demonstrating game logic architecture, data flow, and production-readiness.

- **Architecture:** MVVM with ViewModel + LiveData; ViewBinding
- **Stack:** Kotlin · Coroutines · Jackson · AndroidX · Material
- **Highlights:** Complex stateful game logic managed via ViewModels; match record persistence; regional rule variations; Google Play published title
- **Published:** [palace-v2 on Google Play](https://play.google.com/store/apps/details?id=com.felipimatheuz.palace)

---

### [File Utils — Java](https://github.com/FelipiMatheuz/file-utils-java)
A focused Java utility library for JSON-based file import/export operations. Demonstrates clean API design, Maven packaging, and cross-platform Java engineering.

- **Stack:** Java · Maven · JSON
- **Highlights:** Reusable library design; clear separation between serialization and file I/O concerns
